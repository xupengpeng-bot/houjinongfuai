# Archive

Status: active
Audience: PM and contributors
Purpose: keep the live doc tree clean without deleting project history.

## What belongs here

- duplicate prototype docs
- historical task sheets
- exploratory CRUD guidance that no longer matches the live model
- superseded contract notes

## What does not belong here

- current scope baseline
- live frontend/backend contract docs
- active team handoff workflow

## Rule

If a document is useful for traceability but unsafe as current implementation truth, archive it.
