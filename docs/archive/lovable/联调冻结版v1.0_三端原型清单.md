## 对象承载边界

- **泵/阀**：设备台账子类型（`devices` 表，通过 `device_type_id` 区分），机井详情页"关联设备"页签展示
- **泵阀关系**：独立配置页 `/ops/pump-valve`，属规则层
- **无独立泵页面或阀页面**
