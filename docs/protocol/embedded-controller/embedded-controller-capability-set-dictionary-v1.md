# 嵌入式控制器能力集字典 v1

## 1. 目的

这份文档用于把“产品模板级能力集”冻结成正式字典，供：

- 新产品选型
- 设备类型定义
- AI 实现任务书
- 平台规则配置

共同使用。

## 2. 字典原则

- 能力集 code 一旦启用，不随意改名
- 说明文案可以调整，code 保持稳定
- 每个能力集都必须明确支付锚点、结算主体、计量基准

## 3. 能力集字典

| code | 名称 | 控制锚点 | 结算主体 | 支付锚点 | 计量基准 | 说明 |
| --- | --- | --- | --- | --- | --- | --- |
| `CS-01` | 泵付费电量型 | `pump_anchor` | `pump_subject` | `pump_controller` | `ENERGY` | 单泵或泵阀一体，按电量收费 |
| `CS-02` | 泵付费时长型 | `pump_anchor` | `pump_subject` | `pump_controller` | `TIME` | 单泵或泵阀一体，按时长收费 |
| `CS-03` | 阀付费时长型 | `valve_anchor` | `valve_subject` | `valve_controller` | `TIME` | 一泵多阀，按阀时长收费 |
| `CS-04` | 阀付费水量型 | `valve_anchor` | `valve_subject` | `valve_controller` | `FLOW` | 一泵多阀，按阀水量收费 |
| `CS-05` | 泵支撑统计型 | `pump_anchor` | `shared_support_subject` | `none` | `N/A` | 泵只做调压、联动、总量统计 |

## 4. 使用规则

- 新产品必须先从这 5 套能力集中选主模板。
- 若无法归类，才能定义临时扩展能力集。
- 新增能力集时，必须同步更新：
  - 本字典
  - 产品模板选型单
  - 能力集规划方案

## 5. 临时扩展格式

若必须新增，可按：

- `CS-XX`

格式临时扩展，并补齐：

- 控制锚点
- 结算主体
- 支付锚点
- 计量基准
- 适用场景
