# 嵌入式控制器硬件能力定义模板 v1

## 1. 目的

这份模板用于把原理图转换成“可操作的硬件能力”。

AI 或开发同学不能直接只看原理图名词实现业务逻辑，必须先经过这一步：

- 原理图资源
- 可用硬件能力
- 能力限制

## 5. 约束与限制

必须明确写出：

- 哪些能力不能并发
- 哪些资源共总线
- 哪些采样频率不能过高
- 哪些继电器不能长期拉住
- 哪些接口只能做统计，不能做结算
