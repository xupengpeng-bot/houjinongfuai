Status: closed
Audience: Cursor
Purpose: freeze the backend Phase 1 skeleton for project blocks, metering points, network-model objects, permission grain, and solver-service contracts before larger frontend cockpit work begins.

## 1. 当前阶段

- Phase 1
- 主骨架冻结阶段

1. `project_block`
2. `metering_point`
3. `network_model` / `network_model_version`
4. `network_node` / `network_pipe`
5. 数据权限主颗粒度：项目级、区块级
6. `solver service` 最小后端 contract 骨架

## 4. 冻结模型

### 4.1 project_block

最小字段：

- `id`
- `block_code`
- `project_id`
- `block_name`
- `center_latitude`
- `center_longitude`
- `boundary_geojson`
- `area_size`
- `priority`
- `default_metering_point_id`
- `status`
- `remarks`

规则：

- `block_code` 系统生成
- 项目中的区块数量由统计得出，不单独存储人工填写字段

### 4.2 metering_point

最小字段：

- `id`
- `metering_point_code`
- `project_id`
- `block_id`
- `asset_id`
- `primary_meter_device_id`
- `metering_type`
- `tariff_plan_id`
- `allocation_rule_id`
- `status`
- `remarks`

规则：

- 变压器台区归资产域
- 国网电表归设备域
- 核算以 `metering_point` 为正式对象

### 4.3 network_model

最小字段：

- `network_model.id`
- `network_model.project_id`
- `network_model.model_name`
- `network_model.source_type`
- `network_model.status`

- `network_model_version.id`
- `network_model_version.network_model_id`
- `network_model_version.version_no`
- `network_model_version.is_published`
- `network_model_version.source_file_ref`

- `network_node.id`
- `network_node.version_id`
- `network_node.node_code`
- `network_node.node_type`
- `network_node.asset_id`
- `network_node.latitude`
- `network_node.longitude`
- `network_node.altitude`

- `network_pipe.id`
- `network_pipe.version_id`
- `network_pipe.pipe_code`
- `network_pipe.pipe_type`
- `network_pipe.from_node_id`
- `network_pipe.to_node_id`
- `network_pipe.length_m`
- `network_pipe.diameter_mm`

### 4.4 数据权限骨架

最小字段：

- `data_scope_policy.id`
- `subject_type`
- `subject_id`
- `scope_type`
- `project_id`
- `block_id`
- `effect`

规则：

- 默认主颗粒度：项目级、区块级
- 不把设备级作为全局默认权限主模型
- 设备级只预留例外能力

### 4.5 solver contract skeleton

最小 contract：

- `POST /ops/solver/preview`
- `POST /ops/solver/plan`
- `POST /ops/solver/explain`
- `POST /ops/solver/simulate`

当前只做 request / response skeleton，不做复杂求解内核。
