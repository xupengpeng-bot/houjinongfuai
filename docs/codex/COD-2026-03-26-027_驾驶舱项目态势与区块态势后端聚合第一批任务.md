# COD-2026-03-26-027 驾驶舱项目态势与区块态势后端聚合第一批任务

Status: closed
Audience: Cursor
Purpose: deliver the first larger-granularity backend aggregation wave for the cockpit so the next frontend wave can stop relying on placeholder cards.

## 1. 当前阶段

- Phase 1
- 驾驶舱真实数据第一批
- 当前目标：一次补齐项目态势与区块驾驶舱的最小后端聚合 contract

## 2. 任务目标

整批交付以下后端能力：

1. `project overview`
   - 项目态势卡片聚合接口
   - 返回项目级核心统计
2. `block cockpit`
   - 区块驾驶舱卡片聚合接口
   - 返回区块级运行与告警摘要
3. options / search
   - 支持按项目、区块查询
   - 支持最小搜索 / 过滤参数
4. 为前端下一批真实驾驶舱接线提供稳定 contract

## 3. 范围

允许做：

- controller / service / dto
- 聚合查询
- mock / seed 对应最小样例
- build / test
- docs / RESULT 回写

不允许做：

- 修改前端业务代码
- 重开架构
- 扩到 solver 真实求解
- 扩到 DWG / 管网模型编辑器

## 4. 建议 contract

### 4.1 `GET /api/v1/ops/project-overview`

最小返回建议：

- `project_count`
- `block_count`
- `active_well_count`
- `online_metering_point_count`
- `running_session_count`
- `open_alert_count`
- `open_work_order_count`

### 4.2 `GET /api/v1/ops/block-cockpit`

支持：

- `project_id`
- `q`

最小返回建议每条：

- `block_id`
- `block_code`
- `block_name`
- `project_id`
- `project_name`
- `status`
- `running_well_count`
- `total_well_count`
- `today_usage_m3`
- `open_alert_count`
